# CodeAlpha_Student-Grade-Tracker# CodeAlpha Student Grade Tracker

## Project Description

This is a Java-based Student Grade Tracker developed as part of the CodeAlpha Internship. The application allows users to add student names and marks, view all student records, calculate the average marks, and find the highest and lowest scores. It is a simple console-based application that demonstrates the use of Java programming concepts.

## Features

- Add student names and marks
- View all student records
- Calculate average marks
- Find the highest score
- Find the lowest score
- Menu-driven console interface

## Technologies Used

- Java
- ArrayList
- Object-Oriented Programming (OOP)
- VS Code

## How to Run

1. Open the project in VS Code.
2. Compile the program.
3. Run the Java application.
4. Use the menu to add students and view results.
